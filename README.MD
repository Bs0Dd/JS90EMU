# JS90EMU (JavaScript MK90 EMUlator)

![JS90](https://raw.githubusercontent.com/Bs0Dd/JS90EMU/master/bitmaps/js90.png)

**JS90EMU** is an emulator of the rare Soviet **microcalculator**/**microcomputer** named **Elektronika MK90**.

Based on a heavily modified version of [**JS85EMU**](https://github.com/Bs0Dd/JS85EMU).
